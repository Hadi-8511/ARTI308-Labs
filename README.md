##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the *Student Performance Dataset* to understand factors affecting student outcomes and to prepare the data for a **classification problem**.

The main goal is to analyze relationships between academic and behavioral features and classify students into performance categories.

##  Dataset Description
The dataset contains student-related information including:
* Demographics (Age, Gender, Ethnicity)
* Academic behavior (StudyTimeWeekly, Absences, Tutoring)
* Activities (Sports, Music, Volunteering, Extracurricular)
* Academic performance (GPA, GradeClass)
